# AI-Fitness-Pro
AI Fitness Pro is an intelligent fitness planning web application that generates personalized workout and health plans using AI. Built with modern technologies, it helps users achieve their fitness goals efficiently through smart recommendations and interactive planning tools.

# Features

- AI-generated personalized workout plans
- Goal-based fitness recommendations
- Multi-page Streamlit interface
- Secure API key management
- Clean and structured UI

- # Tech Stack

- Python
- Streamlit
- Google Gemini API
- dotenv

├── main_page.py

│   ├── 1_User_Input.py
│   ├── 2_AI_Planner.py

# How It Works

- User enters fitness details.
- Data is formatted into a structured prompt.
- Gemini API generates a workout plan.
- Plan is displayed in a clean format.
